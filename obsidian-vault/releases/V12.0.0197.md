# V12.0.0197

## Objetivo

Corrigir a segunda falha de compilacao observada apos a `V12.0.0196`.

## Alteracoes

- removida a tipagem local `As eTipoEvento` em `Svc_Rodizio.AvancarFila`
- mantidos os membros `EVT_*` como constantes do projeto
- preservada toda a logica de atomicidade minima, rollback, snapshot V2 e `ATM_001`

## Causa raiz

Neste projeto, o compilador VBA esta aceitando os membros do enum como constantes globais, mas nao esta resolvendo de forma confiavel o tipo `eTipoEvento` fora do modulo `Audit_Log`.

## Validacao pendente

1. Importar a `V12.0.0197` no Excel e compilar.
2. Rodar `CT2_ExecutarSmokeRapido`.
3. Confirmar `ATM_001 = OK`.
4. Confirmar criacao das abas `SNAPV2_*`.
