# V12.0.0075 — Interface: botão “Credencia Empresa” volta a responder (binding por Caption)

## Resumo

Corrige o problema em que o operador clica em **Credencia Empresa** e “nada acontece”. Em algumas exportações/importações, o nome interno do botão muda no `.frx`, quebrando o handler padrão (`NomeDoControle_Click`).  
Nesta versão, o Menu liga o clique via `WithEvents`, localizando o botão pelo **Caption** e encaminhando para o fluxo de credenciamento.

## Alterações

- `vba_export/Menu_Principal.frm`
  - Adiciona `mBtnCredenciarEmpresa` (`WithEvents`) e liga no `UserForm_Initialize` via busca por Caption (“CREDENCIA”).
  - `mBtnCredenciarEmpresa_Click` encaminha para `Credencia_Empresa_Click`.
