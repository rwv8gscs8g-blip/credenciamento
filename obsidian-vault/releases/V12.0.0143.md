# V12.0.0143

## Objetivo

Corrigir definitivamente `ResetarECarregarCNAE_Padrao` — a importacao de CNAEs que deixava ATIVIDADES vazia.

## Causa-raiz

A importacao falhava silenciosamente por excesso de complexidade encadeada:

1. **Protecao prematura**: `Util_RestaurarProtecaoAba` era chamada ENTRE o clear e o import, reprotegendo a aba antes de gravar os dados.
2. **ProximoId por-linha**: cada registro chamava `ProximoId(SHEET_ATIVIDADES)` que fazia seu proprio ciclo `Util_PrepararAbaParaEscrita` / `Util_RestaurarProtecaoAba` — centenas de ciclos de protecao durante o import.
3. **AtividadeJaExiste O(n²)**: cada registro verificava duplicidade varrendo todas as linhas existentes (desnecessario apos reset total).
4. **Dados apagados ANTES de validar CSV**: se `LocalizarArquivoCnaePadrao` retornava "" (CSV nao encontrado), os dados ja estavam destruidos.
5. **Erro silencioso**: `On Error GoTo erro_carregamento` capturava erros mas nao indicava em qual etapa ocorreu.

## Alteracoes

### Preencher.bas — `ResetarECarregarCNAE_Padrao` (reescrita completa)

A funcao foi reescrita com 9 etapas numeradas e diagnostico em cada uma:

1. **Localizar CSV** — `LocalizarArquivoCnaePadrao()`. Se nao encontrado, exibe `ThisWorkbook.Path` e todos os caminhos tentados.
2. **Ler e validar conteudo** — `LerTextoArquivo(caminho)`. Se vazio, aborta SEM apagar dados.
3. **Parsear cabecalho** — detecta delimitador (`;` ou `,`), identifica colunas CNAE/DESCRICAO. Fallback posicional se cabecalho nao reconhecido.
4. **Confirmar com usuario** — MsgBox mostra caminho, total de linhas, delimitador e colunas detectadas.
5. **Preparar aba** — desprotege UMA VEZ. Nao reprotege ate o final.
6. **Limpar dados** — `ClearContents` + reset contador.
7. **Import direto inline** — escreve `wsAtiv.Cells()` diretamente:
   - Contador sequencial inline (`contadorId`) em vez de `ProximoId` por linha.
   - Sem `AtividadeJaExiste` (aba acabou de ser limpa).
   - Tratamento de descricoes com virgula (reconstroi campo quando Split quebrou aspas).
   - `Application.StatusBar` + `DoEvents` a cada 50 registros para feedback visual.
8. **Restaurar protecao** — uma unica chamada apos todas as escritas.
9. **Pos-processamento** — `InvalidarCacheCnaeAtividade`, `LimparCadServParaAssociacaoManual`, listas, save.

Error handler melhorado: mostra etapa, Err.Number, Err.Description, qtd importados ate o erro, caminho CSV.

### Preencher.bas — `LocalizarArquivoCnaePadrao`

Agora busca tambem no diretorio-pai de `ThisWorkbook.Path` (6 caminhos no total), cobrindo workbook em subpasta.
