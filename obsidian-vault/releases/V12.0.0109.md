# V12.0.0109 - Release Notes

**Data de Liberacao**: 2026-04-10
**Status**: Candidata - aguardando importacao e compilacao no Excel
**Base Estavel**: V12.0.0107

---

## Resumo

Pacote de estabilizacao do processo de release e compilacao. Esta iteracao nao prioriza funcionalidade nova; o foco e reduzir regressao entre IAs, centralizar metadata de release em codigo versionado e alinhar o pacote `vba_import/` com a fonte de verdade `vba_export/`.

---

## O que mudou

### 1. Metadata centralizada da release
- Novo modulo `vba_export/App_Release.bas`
- Versao, status da release e URLs oficiais do projeto saem do `Menu_Principal.frm`
- `Menu_Principal` passa a consumir funcoes centralizadas

### 2. Endurecimento do Menu_Principal
- Removidas constantes locais de release e URLs
- Corrigidas 2 ocorrencias do killer pattern `Dim ... : ... =`
- Botao "Sobre" passa a refletir a metadata centralizada

### 3. Estabilizacao do pacote de importacao
- `scripts/publicar_vba_import.sh` atualizado para incluir `App_Release.bas`
- `vba_import/000-ORDEM-IMPORTACAO.txt` realinhado com o pacote gerado
- Contagem de modulos e ordem de importacao atualizadas

### 4. Normalizacao de governanca
- `obsidian-vault/regras/Governanca.md` atualizado para fluxo `candidata -> compilada -> publicada`
- `obsidian-vault/regras/Compilacao-VBA.md` atualizado para regra de `change-set coeso`
- `obsidian-vault/01-CONTEXTO-IA.md` e `obsidian-vault/handoff/Prompt-Codex.md` alinhados com o novo fluxo
