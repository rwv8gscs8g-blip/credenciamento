# V12.0.0135

## Objetivo

Tornar a base de CNAEs estrutural, persistente no workbook e sem fallback silencioso para dados ficticios.

## Alteracoes

- `Preencher.bas`
- `CargaInicialCNAE_SeNecessario` deixa de criar atividades padrao ficticias quando a aba `ATIVIDADES` estiver vazia.
- Quando a fonte canonica existir, a carga passa a:
  - importar os CNAEs reais
  - sincronizar descricoes de `CAD_SERV`
  - salvar o workbook automaticamente para persistir a baseline estrutural
- `GarantirAtividadesBase` passa a atuar como guarda estrutural e alerta o administrador quando a baseline canonica estiver ausente.
- `ImportarCNAE_Padrao_TimeAI` e `CargaCNAE_Permanente` agora tambem sincronizam `CAD_SERV` e persistem a planilha com salvamento seguro.

- `Auto_Open.bas`
- Ao abrir o workbook, o sistema tenta garantir a carga estrutural de CNAE antes de exibir o menu principal.

- `App_Release.bas`
- Versao atualizada para `V12.0.0135`.

## Impacto esperado

- Os CNAEs reais passam a ser carregados uma vez e mantidos no workbook.
- Usuários em outras maquinas nao dependem do CSV na raiz, desde que recebam a planilha ja persistida com a baseline estrutural.
- A aplicacao deixa de mascarar a ausencia da base canônica com atividades ficticias.
