# V12.0.0199

## Objetivo

Atacar a causa recorrente das falhas de compilacao apos reimportacao: projeto VBA incompleto ou com modulo obrigatorio ausente.

## Alteracoes

- `Importador_VBA` ganhou `Verificar_ModulosObrigatorios`
- `Importador_VBA` ganhou `AAA_Verificar_Estabilizacao_PosImportacao`
- fluxo de importacao agora orienta validar modulos obrigatorios antes de compilar

## Causa raiz tratada

No erro atual, a captura mostra `Repo_Avaliacao` ausente no projeto VBA, o que quebra `Svc_Avaliacao` em `RepoAvaliacaoInserir`. A `0199` nao mascara isso; ela passa a denunciar a estrutura incompleta imediatamente apos a importacao.

## Validacao pendente

1. Importar `Repo_Avaliacao.bas` no workbook atual.
2. Importar `Importador_VBA.bas` e `App_Release.bas`.
3. Rodar `Verificar_ModulosObrigatorios`.
4. Rodar `Verificar_SemDuplicidade`.
5. Compilar o projeto.
