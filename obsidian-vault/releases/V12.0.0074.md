# V12.0.0074 — Interface: credenciar empresa mais robusto (fallback por CNPJ/Razão)

## Resumo

Corrige o fluxo de credenciamento pela interface quando o `ListBox` de empresas não fornece o ID corretamente: o botão **Credencia Empresa** agora tenta localizar o `EMP_ID` na aba `EMPRESAS` usando **CNPJ** (ou, como fallback, **Razão Social**) antes de abrir o formulário de credenciamento.

Isso é pré‑requisito para validação visual do rodízio, pois sem credenciamento não há empresas disponíveis para emitir Pré‑OS.

## Alterações

- `vba_export/Menu_Principal.frm`
  - `Credencia_Empresa_Click`: fallback de `EMP_ID` via busca na aba `EMPRESAS` (CNPJ/Razão).
