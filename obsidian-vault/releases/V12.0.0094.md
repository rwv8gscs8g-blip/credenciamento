# V12.0.0094 — Estabilização final (CNAE + UI + Bateria)

## Resumo
- **CNAE permanente**: adicionada rotina de importação da base normalizada de CNAE (serviços) para a aba `ATIVIDADES`, com tentativa automática na inicialização (não bloqueante).
