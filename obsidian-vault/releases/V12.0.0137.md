# V12.0.0137

## Objetivo

Fechar o passo administrativo de reinstalacao da baseline estrutural permanente de CNAE com salvamento seguro e mensagem explicita de sobrescrita.

## Alteracoes

- `Preencher.bas`
- `ResetarECarregarCNAE_Padrao` agora:
  - informa explicitamente que os dados atuais de `ATIVIDADES` podem ser substituidos
  - recarrega a lista de atividades apos a importacao
  - salva o workbook com `Util_SalvarWorkbookSeguro`
  - orienta o operador a usar `Ctrl+S` se o salvamento automatico falhar

- `App_Release.bas`
- Versao atualizada para `V12.0.0137`.

## Impacto esperado

- O administrador passa a ter um passo unico e seguro para reinstalar a baseline estrutural de CNAE.
- A planilha resultante pode ser salva e distribuida ja com os CNAEs permanentes persistidos.
