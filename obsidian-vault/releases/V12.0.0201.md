Data: 2026-04-19
Status: EM_VALIDACAO
Autor: GPT-5 (Codex)

## Objetivo

Endurecer o `Importador_VBA` contra projetos VBA ja contaminados, inclusive quando o proprio identificador nativo `Collection` entra em conflito durante a compilacao.

## Alteracoes

- O importador deixou de tipar listas internas como `Collection`
- A lista de itens resolvidos no incremental passou a usar array dinamico simples
- A lista de variantes de nomes fantasmas passou a usar array fixo simples
- A ordem deterministica foi preservada, incluindo `Mod_Types` primeiro

## Impacto esperado

- Maior chance de o importador compilar mesmo em workbooks que ja tenham colisao com nomes nativos
- Menor dependencia da higiene previa do projeto para conseguir rodar o fluxo de estabilizacao
