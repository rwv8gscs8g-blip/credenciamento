# V12.0.0069 — Interface: Pre-OS mais resiliente + Sobre/GitHub + impressão padrão + preservar CAD_SERV no “Zerar base”

## Resumo

Correções e ajustes focados em **operabilidade na interface**: emissão de Pre‑OS mais tolerante a estado de tela, “Sobre” com texto mais claro (rodízio) e sem abrir GitHub no OK, botão **GitHub** dedicado, impressão usando **impressora padrão do Windows** (sem prompt de setup) e política de “Zerar base” preservando também `CAD_SERV` (além de `ATIVIDADES/CNAE`).

## Alterações

- `vba_export/Menu_Principal.frm`
  - `B_PreOS_Click`: valida lista de serviços carregada; leitura defensiva do item selecionado; em caso de ausência de dados para template, não trava — informa e orienta a consultar `PRE_OS`.
  - “Sobre”: reescrito para descrever o sistema como **controle de rodízio**; OK apenas fecha.
  - Botão **GitHub**: abre o repositório; “Sobre” passa a exibir também o link das notas de versão.
  - Relatórios: impressão direta via `PrintOut` usando a **impressora padrão** (sem `xlDialogPrinterSetup`).
  - `RELEASE_ATUAL` / `ITERACAO_ATUAL`: **V12.0.0069** / **12069**.

- `vba_export/Preencher.bas`
  - `Limpa_Base`: preserva `CAD_SERV` (e continua preservando `ATIVIDADES/CNAE`) ao zerar a base operacional.
