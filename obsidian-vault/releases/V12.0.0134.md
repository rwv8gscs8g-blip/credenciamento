# V12.0.0134

## Objetivo

Estabilizar a abertura do cadastro de servicos para evitar desvio indevido para formularios de entidade em workbooks com instancias residuais.

## Alteracoes

- `Menu_Principal.frm`
- O handler `Cad_Servico_Click` agora:
  - descarta instancias visiveis residuais de `Altera_Entidade` e `Cadastro_Servico`
  - recarrega a lista de atividades pelo modulo `Preencher`
  - abre uma nova instancia modal de `Cadastro_Servico` via `VBA.UserForms.Add`
  - exibe mensagem explicita em caso de falha na abertura

- `App_Release.bas`
- Versao atualizada para `V12.0.0134`.

## Impacto esperado

- O botao `Cadastrar Servico` passa a abrir de forma deterministica o formulario correto.
- Reduz risco de drift entre instancias predefinidas e formularios residuais no workbook importado.
