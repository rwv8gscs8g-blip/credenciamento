# V12.0.0146 — Correcao CAD_SERV com Dados Duplicados

## Resumo

Correcao do problema onde as telas "Altera Cadastro de Servicos" e "Atribuicao de Empresa para Servico" exibiam todas as 612 atividades como servicos duplicados (Nome Atividade = Nome Servico). A causa raiz era dados lixo na aba CAD_SERV herdados da base V138.

## Causa Raiz

A aba CAD_SERV continha 612 registros espelhando ATIVIDADES (cada atividade copiada como servico com descricao identica, IDs todos "001"). Esses dados vieram da base V138 restaurada e nunca foram limpos. As funcoes PreencherManutencaoValor, PreenchimentoServico e PreenchimentoCRServico estavam corretas — leem de CAD_SERV como esperado.

## Correcao

### Importar_Agora.bas
- Adicionada limpeza automatica de CAD_SERV apos importacao de CNAEs
- Nova macro `LIMPAR_CADSERV_AGORA` para limpeza manual independente
- Nova sub privada `LimparCadServAposImportacao` que desprotege, limpa dados e reprotege

### Procedimento para Corrigir Planilha Atual
1. Abrir a planilha
2. Alt+F8 > Executar `LIMPAR_CADSERV_AGORA`
3. Confirmar a limpeza
4. CAD_SERV ficara vazia — cadastrar servicos manualmente via "Cadastrar Servico"
