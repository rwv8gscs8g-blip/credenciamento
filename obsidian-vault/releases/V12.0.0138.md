# V12.0.0138

## Objetivo

Corrigir a importacao da coluna CNAE no CSV canônico e alinhar `CAD_SERV` com a baseline estrutural de `ATIVIDADES`.

## Alteracoes

- `Preencher.bas`
- O importador CSV deixa de tratar o cabecalho `ID` como coluna de CNAE.
- `ResetarECarregarCNAE_Padrao` agora recria `CAD_SERV` a partir de `ATIVIDADES`, apagando os servicos antigos e gerando uma baseline coerente com a estrutura nova.
- A rotina passa a recarregar tambem a tela de manutencao de servicos apos o reset.

- `App_Release.bas`
- Versao atualizada para `V12.0.0138`.

## Impacto esperado

- A aba `ATIVIDADES` volta a preencher a coluna `CNAE` com o codigo correto do CSV.
- A tela `Altera Cadastro de Servicos` passa a refletir a baseline atualizada, em vez de manter os 3 servicos antigos.
